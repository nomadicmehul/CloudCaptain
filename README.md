## What is CloudCaptain?

CloudCaptain is a community-driven, open-source learning platform with **476+ PDFs/books**, **50+ documentation pages**, **8 career paths**, and **7 structured learning roadmaps** — all free, forever.

Whether you're a complete beginner or a seasoned professional, CloudCaptain has something for you.

## Website

The CloudCaptain learning platform is built with Docusaurus and deployed to GitHub Pages.

```bash
cd website
npm install
npm start         # Dev server at localhost:3000
npm run build     # Production build
```

## What's Inside

| Category | Content |
|:---------|:--------|

## Career Paths (AI Era)

1. **DevOps Engineer** — Automate software delivery
2. **Cloud Engineer / Architect** — Design cloud infrastructure at scale
3. **Platform Engineer** — Build internal developer platforms
4. **AI/ML Infrastructure Engineer** — Run AI workloads at scale
5. **Site Reliability Engineer** — Keep the internet running
6. **Linux Systems Master** — Deep systems expertise
7. **Cloud Security / DevSecOps Engineer** — Secure everything
8. **FinOps Practitioner** — Optimize cloud costs
