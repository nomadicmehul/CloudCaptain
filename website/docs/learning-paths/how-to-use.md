# How to Use CloudCaptain

## Navigation

CloudCaptain is organized into three main sections:

### Learning Paths
Structured roadmaps that take you from beginner to advanced. Each path includes recommended resources, estimated timelines, and hands-on exercises. Start here if you're unsure where to begin.

### Tools & Tech
Deep dives into specific technologies. Each section includes concepts, commands, cheatsheets, interview prep, and curated external resources. Use this when you want to learn a specific tool.

### Cloud Providers
Provider-specific guides for AWS, Azure, GCP, and DigitalOcean. Includes architecture guides, exam prep, and best practices.

## Recommended Approach

1. **Pick a Learning Path** that matches your goals
2. **Follow the roadmap** step by step — don't try to learn everything at once
3. **Practice hands-on** — use the exercises and labs in each section
4. **Use cheatsheets** as quick references while working
5. **Prepare for interviews** when you're ready to level up
