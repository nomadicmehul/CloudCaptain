# DevOps Learning Path

A structured roadmap from beginner to advanced DevOps practitioner.

:::tip Don't try to learn everything at once
Focus on one topic at a time. The goal is to say "I know what DevOps is and I can build anything with it." Learn the rest on demand.
:::

## Stage 1: Foundations (Weeks 1-4)

| Topic | What to Learn | Resources |
|:------|:-------------|:----------|
| **Version Control** | [Git](/docs/tools/git/) | Git is used everywhere — master it first |
| **Programming** | [Python](/docs/tools/python/) or Go | Every DevOps engineer needs at least one language |
| **Operating System** | [Linux](/docs/tools/linux/) | The backbone of cloud infrastructure |
| **Networking** | [Networking Basics](/docs/tools/networking/) | DHCP, DNS, TCP/IP, HTTP, SSH |
| **Scripting** | [Bash](/docs/tools/bash/) | Automate everything from day one |

## Stage 2: Core DevOps (Weeks 5-10)

| Topic | What to Learn | Resources |
|:------|:-------------|:----------|
| **Containers** | [Docker](/docs/tools/docker/) | Containerize applications |
| **CI/CD** | [Jenkins](/docs/tools/jenkins/), [GitHub Actions](/docs/tools/github-actions/) | Automate build, test, deploy |
| **IaC** | [Terraform](/docs/tools/terraform/) | Infrastructure as Code |
| **Config Mgmt** | [Ansible](/docs/tools/ansible/) | Configuration management at scale |
| **Cloud** | [AWS](/docs/cloud/aws/) or [Azure](/docs/cloud/azure/) | Pick one cloud provider to start |

## Stage 3: Advanced (Weeks 11-16)

| Topic | What to Learn | Resources |
|:------|:-------------|:----------|
| **Orchestration** | [Kubernetes](/docs/tools/kubernetes/) | Container orchestration |
| **Helm** | [Helm Charts](/docs/tools/helm/) | Package management for K8s |
| **GitOps** | ArgoCD, FluxCD | Declarative deployments |
| **Monitoring** | [Prometheus & Grafana](/docs/tools/observability/) | Observability stack |
| **Security** | [DevSecOps](/docs/tools/devsecops/) | Shift-left security |

## Stage 4: Expert (Ongoing)

| Topic | Description |
|:------|:-----------|
| **SRE Practices** | SLOs, SLIs, error budgets, incident management |
| **Platform Engineering** | Internal developer platforms, golden paths |
| **FinOps** | Cloud cost optimization |
| **Multi-Cloud** | Architecting across cloud providers |
| **Chaos Engineering** | Resilience testing in production |

## Recommended Certifications

- AWS Certified DevOps Engineer Professional
- Certified Kubernetes Administrator (CKA)
- HashiCorp Certified: Terraform Associate
- Azure DevOps Engineer Expert
- Google Professional Cloud DevOps Engineer
