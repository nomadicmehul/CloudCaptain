# Amazon Web Services (AWS)

The world's most comprehensive and broadly adopted cloud platform, offering 200+ fully featured services.

## Core Services to Master

| Category | Services |
|:---------|:---------|
| **Compute** | EC2, Lambda, ECS, EKS, Fargate |
| **Storage** | S3, EBS, EFS, Glacier |
| **Databases** | RDS, DynamoDB, ElastiCache, Aurora |
| **Networking** | VPC, ELB, Route 53, CloudFront |
| **Security** | IAM, KMS, WAF, GuardDuty |
| **DevOps** | CodePipeline, CodeBuild, CodeDeploy, CloudFormation |
| **Monitoring** | CloudWatch, X-Ray, CloudTrail |

## AWS Certification Path

| Level | Certification |
|:------|:-------------|
| Foundational | AWS Cloud Practitioner (CLF-C02) |
| Associate | Solutions Architect, Developer, SysOps Admin |
| Professional | Solutions Architect Pro, DevOps Engineer Pro |
| Specialty | Security, Networking, Data Analytics, ML |

## Architecture Best Practices

Follow the [AWS Well-Architected Framework](https://aws.amazon.com/architecture/well-architected/):
- **Operational Excellence** — Automate operations
- **Security** — Protect data and systems
- **Reliability** — Recover from failures
- **Performance Efficiency** — Use resources efficiently
- **Cost Optimization** — Eliminate waste
- **Sustainability** — Minimize environmental impact
