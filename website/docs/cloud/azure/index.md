# Microsoft Azure

Microsoft's cloud platform with 200+ products and services for building, running, and managing applications.

## Core Services

| Category | Services |
|:---------|:---------|
| **Compute** | VMs, AKS, Functions, App Service |
| **Storage** | Blob, Files, Disks, Data Lake |
| **Databases** | SQL Database, Cosmos DB, Cache for Redis |
| **Networking** | VNet, Load Balancer, Front Door, DNS |
| **DevOps** | Azure DevOps, GitHub Actions |
| **AI** | OpenAI Service, Cognitive Services, ML |

## Certification Path

1. **AZ-900** — Azure Fundamentals
2. **AZ-104** — Azure Administrator
3. **AZ-204** — Azure Developer
4. **AZ-400** — Azure DevOps Engineer
5. **AZ-305** — Azure Solutions Architect
