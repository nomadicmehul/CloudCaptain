## Key Concepts

- **FinOps Lifecycle** — Inform, Optimize, Operate for continuous cost management
- **Cost Visibility** — See spending broken down by service, team, project
- **Right-Sizing** — Match instance sizes to actual workload requirements
- **Reserved Instances** — 1-3 year commitments for 40-60% savings
- **Cost Allocation** — Assign cloud costs to teams via tagging
- **Chargeback Model** — Charge teams for their cloud spend
- **Cloud Cost Optimization** — 20-30% cost reduction typically achievable

## Common Use Cases

- Reduce cloud spending by 20-30% without impacting innovation
- Understand where cloud budget is going
- Allocate costs fairly across teams
- Purchase Reserved Instances at the right time
- Right-size overprovisioned instances
- Create culture of cost awareness
